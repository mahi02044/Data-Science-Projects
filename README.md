# 📊 Sales Dashboard Analysis

## 📌 Project Overview
This project analyzes retail sales data from the Superstore dataset
to uncover business insights around sales, profit, and customer trends
using Python and data visualization.

## 🎯 Objective
- Identify which product categories generate the most sales
- Find which regions are most profitable
- Analyze monthly sales trends over time
- Discover the top 10 most profitable products

## 🛠️ Tools & Libraries Used
- Python
- Google Colab
- Pandas (data cleaning & analysis)
- Matplotlib (data visualization)
- Seaborn (data visualization)

## 📁 Dataset
- **Source:** Kaggle Superstore Dataset
- **Link:** https://www.kaggle.com/datasets/vivek468/superstore-dataset-final
- **Size:** 9,994 rows × 21 columns

## 📊 Key Insights
- Technology category had the highest total sales
- The West region contributed the most to overall profit
- Sales peaked during November and December each year
- Some Furniture products showed negative profit margins

## 📸 Charts Created
1. Total Sales by Category (Bar Chart)
2. Profit Distribution by Region (Pie Chart)
3. Monthly Sales Trend Over Time (Line Chart)
4. Top 10 Most Profitable Products (Bar Chart)
5. Sales vs Profit by Category (Scatter Plot)

## 🚀 How to Run
1. Open the `.ipynb` file in Google Colab
2. Upload `Sample - Superstore.csv` to Colab
3. Run each cell from top to bottom
